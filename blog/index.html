Face Detection Web App

13/04/2020

A web app that allows users to sign in/register and will detect any faces in an image using the Clarifai API. It comprises a React frontend, a Node.js/Express.js backend with a PostgreSQL database and is deployed on Heroku. This was the final project for a web development Udemy course.

AWS Lambda Web Scraper

28/08/20

An AWS Lambda function built using the Serverless framework that runs on a daily cron job. It uses the Cheerio library to scrape Smooth 4000's Souncloud page and checks whether a new song has been uploaded. If a new song is found it then triggers SES to send me an email.
